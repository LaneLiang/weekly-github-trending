# DC-DC Buck 变换器多目标优化指标分档体系

**基准拓扑**：Buck 12V→5V, L=100μH, C=100μF, f_sw=100kHz, R_load=5Ω, ESR=0.05Ω
**物理极限参考**：纹波地板 ~0.37% (ESR主导), LC谐振周期 ~0.6ms, 满载导通损耗 ~2%

---

## 一、分档指标总览

### P0 — 功能可用 (Functional Baseline)

SAC 训练后必须全部达成，否则控制器不可用。

| 指标 | 目标 | 物理依据 |
|------|------|---------|
| 稳态电压误差 | < 0.5% (25mV) | 远高于纹波地板，充裕 |
| 纹波率 | < 2% (100mV) | 纹波地板 0.37% 的 5×，充裕 |
| 过冲 | < 5% (250mV) | 标准工业要求 |
| 下冲 | < 5% (250mV) | 同上 |
| 恢复时间 | < 500μs (50 T_sw) | LC 周期的 ~1×，充裕 |
| 启动时间 | < 10ms | ~15× LC 周期，充裕 |
| 满载 CCM 效率 | > 88% | 导通损耗~2%，开关+磁芯~10% 裕量 |

### P1 — 工程可用 (Engineering Grade)

SAC 训练后应全部达成，代表有竞争力的工程方案。

| 指标 | 目标 | 物理依据 |
|------|------|---------|
| 稳态电压误差 | < 0.2% (10mV) | 接近但高于纹波地板 |
| 纹波率 | < 1% (50mV) | 纹波地板 0.37% 的 2.7× |
| 过冲 | < 2% (100mV) | 良品级瞬态要求 |
| 下冲 | < 2% (100mV) | 同上 |
| 恢复时间 | < 200μs (20 T_sw) | LC 周期的 ~1/3，有挑战 |
| 启动时间 | < 5ms | ~8× LC 周期 |
| 满载 CCM 效率 | > 92% | 总损耗 < 8% |
| 轻载 DCM 效率 (20% load) | > 80% | DCM 开关损耗占比高，合理目标 |

### P2 — 优秀 (Excellent)

达成 P1 基础上追求的 stretch goal，对标高端商用控制器。

| 指标 | 目标 | 物理依据 |
|------|------|---------|
| 稳态电压误差 | < 0.1% (5mV) | 接近纹波地板 |
| 纹波率 | < 0.5% (25mV) | 仅 1.4× 纹波地板，需优化 ESR |
| 过冲 | < 1% (50mV) | 极严格 |
| 下冲 | < 1% (50mV) | 同上 |
| 恢复时间 | < 100μs (10 T_sw) | 仅 10 个开关周期，接近极限 |
| 启动时间 | < 3ms | ~5× LC 周期 |
| 满载 CCM 效率 | > 95% | 总损耗 < 5%，需全链路优化 |
| 轻载 DCM 效率 (20% load) | > 85% | DCM 极限目标 |

## 二、工况定义

### 标准工况（训练/评估默认）
- Vin = 12V, Vref = 5V, R_load = 5Ω (满载 1A)
- 环境温度 25°C（不影响当前模型）
- 从零初始条件启动

### 负载跳变工况（评估瞬态）
- Load step: 50% ↔ 100% (R_load: 10Ω ↔ 5Ω)
- Load step: 20% ↔ 80% (R_load: 25Ω ↔ 6.25Ω)

### 轻载 DCM 工况（评估 DCM 效率）
- R_load = 25Ω (20% 满载, 0.2A)
- R_load = 50Ω (10% 满载, 0.1A)

### 输入电压变化工况（评估线性调整率）
- Vin = 10.8V (−10%)
- Vin = 13.2V (+10%)

## 三、多目标权重设计原则

各指标之间存在物理 trade-off，LLM 教练的职责是在训练过程中动态调整权重来导航 Pareto 前沿。

### 权重映射策略

```
用户语义偏好 → LLM 翻译 → 权重向量 [w_ev, w_vr, w_eff, w_tr, w_os, w_us, w_ts]

示例：
"效率优先"   → w_eff=2.0, w_ev=1.0, w_vr=0.5, w_tr=0.3, w_os=0.5, w_us=0.5, w_ts=0.3
"瞬态优先"   → w_tr=2.0, w_os=2.0, w_us=2.0, w_ev=0.5, w_vr=0.3, w_eff=0.3, w_ts=0.5
"低纹波优先" → w_vr=2.0, w_ev=1.0, w_eff=0.5, w_tr=0.3, w_os=0.5, w_us=0.5, w_ts=0.3
"综合均衡"   → 所有权重 = 1.0
```

### 训练阶段权重调度

| 阶段 | Episode | 策略 | 主导权重 |
|------|---------|------|---------|
| Startup | 0-30 | 优先让 SAC 稳定学到基础控制 | w_ev=1.0, 其他=0.3 |
| Exploration | 30-100 | 引入全部目标，鼓励探索 | 均衡权重 1.0 |
| Refinement | 100-200 | 根据用户偏好聚焦 | LLM 根据偏好动态调整 |

## 四、评估指标

### 单点评估
对每个 trial 的 final policy，在 7 个指标上打分，标注是否达成各档目标。

### 多目标评估
- **Hypervolume (HV)**：以 P0 指标为 reference point，计算非支配解集覆盖的 HV
- **Preference Satisfaction Rate (PSR)**：给定用户偏好文本，LLM 引导的权重搜索是否产生对应的 Pareto 区域偏移
- **Tier Pass Rate**：P0/P1/P2 各档的指标达成率

### 实验基线

| Baseline | 权重策略 | 目的 |
|----------|---------|------|
| B1 | 固定均衡权重 [1,1,1,1,1,1,1] | 无优化的单点 baseline |
| B2 | Random Search 权重 | 传统多目标搜索上限 |
| B3 | LLM Fixed-Interval 权重 | LLM 教练 vs Random |
| B4 | LLM Event-Triggered 权重 | Plateau 感知 vs 固定间隔 |
| B5 | LLM 偏好驱动 ("效率优先") | LLM 独有的语义→权重映射能力 |

## 五、P0 达成即训练成功

训练成功的定义为 **P0 全部指标达成**。在指标体系中：
- P0 是充分考虑了物理裕量的合理目标
- P1 是良好工程方案的合理期待
- P2 是接近物理极限的优秀目标

**不建议**将所有指标都设为物理极限值——那会导致 (a) SAC 无法收敛到任何可行解，(b) 多目标优化失去 Pareto 前沿的意义（前沿退化为一个不存在点）。
